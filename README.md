ok so i actually enjoyed this alot! my process was to first be able to read in ppms and print them out on stdout. then once i did that it was learning jgraph and just trying to get a pdf of a black and white qr code. From there i moved to rgb color then patterns then the picture of planks face but it turned out meh in my opinion. If you want to change the QR content you can do so in the Makefile. I tested all the ones that i provided and they all scan correctly for me. Also the repo listed for the qrcodes is not like an actual generater so i just made my own with their code. 

### To Run
all you have to do is:
- clone my repo
- type make in the terminal
- you should see 6 pdfs show up for the different things i did! (:
- make clean cleans everything up including the downloaded repo.
